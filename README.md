Medical Insurance Cost Prediction

This project predicts medical insurance charges using Linear Regression based on customer details like age, BMI, smoking status, and region.

Dataset

Rows: 1338

Features: age, sex, bmi, children, smoker, region

Target: charges

Model

Algorithm: Linear Regression

Train-Test Split: 80% / 20%

Performance

Training R² Score: 0.75

Testing R² Score: 0.74

Sample Prediction

Input: (31, 1, 25.74, 0, 1, 0)
Predicted Cost: USD 3760.08

Tech Stack

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
